## 1. 向量空间与子空间

| 定义/定理 | 说明                                                                                           |
|----------|----------------------------------------------------------------------------------------------|
| **向量空间** | 集合 $$\(V\)$$ 与标量域 $$\(F\)$$（实数 $$\(\mathbb{R}\)$$ 或复数 $$\(\mathbb{C}\)$$）满足加法与数乘封闭、分配律、结合律、单位元与逆元等五条公理。    |
| **子空间** | $$\(W \subset V\)$$ 若对任意 $$\(u,v \in W\)$$ 与 $$\(\alpha \in F\)$$，都有 $$\(u+v \in W\)$$ 且 $$\(\alpha u \in W\)$$。 |
| **线性无关** | $$\(v_1,\dots,v_k\)$$ 满足 $$\(\sum_{i=1}^k a_i v_i = 0 \Rightarrow a_i=0\)$$。                         |
| **线性相关** | 存在非全为零的系数使得线性组合为 0。                                                                          |
| **基** | 一组既线性无关又张成整个空间的向量。                                                                           |
| **维数** | 任意基的向量个数；记为 $$\(\dim V\)$$。                                                                      |
| **秩-零化定理** | 对矩阵 $$\(A\in F^{m\times n}\)$$，$$\(\operatorname{rank}(A)+\operatorname{nullity}(A)=n\)$$。           |
| **维数公式** | 对子空间 $$\(U,V\subseteq W\)$$，$$\(\dim(U+V)=\dim U+\dim V-\dim(U\cap V)\)$$。                           |
| **投影** | 对非零向量 $$\(a\)$$，投影到 $$\(b\)$$ 上为 $$\(\operatorname{proj}_b a = \frac{a\cdot b}{b\cdot b}\,b\)$$。         |

## 2. 线性映射与矩阵

| 定义/定理 | 说明 |
|----------|------|
| **线性映射** | $$\(T:V\to W\)$$ 满足 $$\(T(u+v)=T(u)+T(v),\;T(\alpha v)=\alpha T(v)\)$$。 |
| **核与像** | $$\(\ker T=\{v\mid T(v)=0\}\)$$，$$\(\operatorname{Im}T=\{T(v)\mid v\in V\}\)$$。 |
| **矩阵与线性映射对应** | 选取基后，$$\(T\)$$ 的矩阵表示为 $$\(A\)$$，且 $$\(T(v)=Av\)$$。 |
| **矩阵乘法** | 若 $$\(A\in F^{m\times n}\)$$, $$\(B\in F^{n\times p}\)$$，则 $$\((AB)_{ij}=\sum_{k=1}^n A_{ik}B_{kj}\)$$。 |
| **逆矩阵** | 若 $$\(A\)$$ 可逆，则存在唯一矩阵 $$\(A^{-1}\)$$ 使 $$\(AA^{-1}=A^{-1}A=I\)$$。逆存在的充要条件是 $$\(\det A\neq 0\)$$。 |
| **行列式** | $$\(\det(A)\)$$ 是 $$\(A\)$$ 的标量，满足 $$\(\det(AB)=\det A\,\det B\)$$，$$\(\det(A^T)=\det A\)$$。 |
| **克莱姆法则** | 对可逆矩阵 $$\(A\)$$，$$\((A^{-1})_{ij}=\frac{C_{ji}}{\det A}\)$$，其中 $$\(C_{ji}\)$$ 为余子式。 |
| **秩** | $$\(\operatorname{rank}A=\dim\operatorname{Im}A=\dim\ker A^T\)$$。 |
| **列/行空间** | $$\(\operatorname{col}A\)$$ 是由 $$\(A\)$$ 的列张成的子空间；$$\(\operatorname{row}A\)$$ 是行张成的子空间。 |

## 3. 行列式的性质与计算

| 性质 | 说明 |
|------|------|
| **加法与数乘** | $$\(\det(\lambda A)=\lambda^n\det A\)$$，$$\(\det(A+B)\neq \det A+\det B\)$$。 |
| **行/列交换** | 交换两行（或两列）会改变符号：$$\(\det(A^{swap})=-\det A\)$$。 |
| **行/列倍加** | 若在某行（列）加上另一行（列）的数倍，则行列式不变。 |
| **展开法则** | 代数余子式展开：$$\(\det A = \sum_{j} a_{ij}C_{ij}\)$$。 |
| **拉普拉斯展开** | 任选一行或一列做展开。 |
| **行列式与逆** | 若 $$\(\det A\neq 0\)$$，则 $$\(A^{-1} = \frac{1}{\det A} \operatorname{adj} A\)$$。 |

## 4. 特征值、特征向量与对角化

| 定义/定理 | 说明 |
|----------|------|
| **特征值与特征向量** | 若存在非零向量 $$\(v\)$$ 使 $$\(Av=\lambda v\)$$，则 $$\(\lambda\)$$ 为 $$\(A\)$$ 的特征值，$$\(v\)$$ 为对应特征向量。 |
| **特征多项式** | $$\(p_A(\lambda)=\det(A-\lambda I)\)$$。根即为特征值。 |
| **几何重数** | $$\(\dim \ker(A-\lambda I)\)$$。 |
| **代数重数** | 在特征多项式中 $$\((\lambda-\lambda_i)^{k}\)$$ 的指数 $$\(k\)$$。 |
| **对角化定理** | 若 $$\(A\)$$ 有 $$\(n\)$$ 个线性无关的特征向量（即 $$\(\sum_i \dim\ker(A-\lambda_i I)=n\)$$），则 $$\(A=PDP^{-1}\)$$，其中 $$\(D\)$$ 为对角矩阵，$$\(P\)$$ 的列为这些特征向量。 |
| **可对角化充要条件** | $$\(A\)$$ 的特征空间维数之和等于 $$\(n\)$$。 |
| **幂法** | 通过 $$\(A^k\)$$ 逼近主特征值。 |
| **正交对角化**（对称/自伴随） | 若 $$\(A\)$$ 为实对称矩阵（或复 Hermitian），则存在正交矩阵 $$\(Q\)$$ 使 $$\(Q^T A Q = D\)$$。 |
| **谱分解** | $$\(A = \sum_{i=1}^r \lambda_i u_i u_i^T\)$$，其中 $$\(u_i\)$$ 为正交特征向量。 |

## 5. 正交与正交投影

| 定义/定理 | 说明 |
|----------|------|
| **内积** | 对 $$\(\mathbb{R}^n\)$$：$$\(u\cdot v = u^T v\)$$。对 $$\(\mathbb{C}^n\)$$：$$\(u^H v\)$$。 |
| **正交** | $$\(u\cdot v=0\)$$。 |
| **正交化**（Gram–Schmidt） | 给定一组线性无关向量 $$\(v_1,\dots,v_k\)$$，生成正交（或正交归一）向量 $$\(u_1,\dots,u_k\)$$。 |
| **正交投影** | 投影矩阵 $$\(P = UU^T\)$$，其中 $$\(U\)$$ 为列为正交基的矩阵。投影满足 $$\(P^2=P,\;P^T=P\)$$。 |
| **正交分解** | $$\(A = Q R\)$$，其中 $$\(Q\)$$ 为正交矩阵，$$\(R\)$$ 为上三角矩阵（QR 分解）。 |
| **最小二乘** | 对过定式不一致的线性方程 $$\(Ax=b\)$$，最优解 $$\(x^* = (A^T A)^{-1} A^T b\)$$。 |

## 6. 线性方程组与解的结构

| 定义/定理 | 说明 |
|----------|------|
| **一致性条件** | $$\(Ax=b\)$$ 有解当且仅当 $$\(b \in \operatorname{col}A\)$$。 |
| **解空间** | 若 $$\(x_0\)$$ 是特解，则所有解为 $$\(x = x_0 + y\)$$，其中 $$\(y\in\ker A\)$$。 |
| **行最简形式** | 用高斯消元得到的最简行阶梯形式可直接读出解。 |
| **可逆性** | 方阵 $$\(A\)$$ 可逆 $$\(\iff\)$$ $$\(\det A \neq 0\)$$，$$\(\iff\)$$ $$\(\operatorname{rank}A=n\)$$。 |

## 7. 线性映射的矩阵表示与变换

| 定义/定理 | 说明 |
|----------|------|
| **基变换** | 若从基 $$\(B\)$$ 变到基 $$\(B'\)$$，则相应矩阵为 $$\(P\)$$，满足 $$\(x_{B'} = P^{-1} x_B\)$$。 |
| **相似变换** | 对方阵 $$\(A\)$$，若存在可逆 $$\(P\)$$ 使 $$\(B=P^{-1}AP\)$$，则 $$\(A\)$$ 与 $$\(B\)$$ 相似，具有相同特征值。 |
| **不变子空间** | 线性变换映射子空间到自身。 |
| **伴随** | 对 $$\(A\)$$，伴随矩阵 $$\(\operatorname{adj}A\)$$ 满足 $$\(A\operatorname{adj}A = \operatorname{adj}A A = \det A\, I\)$$。 |

## 8. 矩阵分解（重要）

| 分解 | 说明 |
|------|------|
| **LU 分解** | $$\(A=LU\)$$，$$\(L\)$$ 为下三角，$$\(U\)$$ 为上三角；需要主元或加倍对角化。 |
| **QR 分解** | $$\(A=QR\)$$，$$\(Q\)$$ 正交，$$\(R\)$$ 上三角；可通过 Gram–Schmidt 或 Householder 变换得到。 |
| **奇异值分解（SVD）** | $$\(A = U \Sigma V^T\)$$，$$\(U,V\)$$ 正交，$$\(\Sigma\)$$ 对角（奇异值）。对任意矩阵都存在。 |
| **Cholesky 分解** | 对正定矩阵 $$\(A\)$$，$$\(A=LL^T\)$$。 |
| **谱分解** | 对对称/自伴随矩阵 $$\(A\)$$，$$\(A = Q\Lambda Q^T\)$$。 |

## 9. 矩阵范数与距离

| 范数 | 说明 |
|------|------|
| **Frobenius 范数** | $$\(\|A\|_F = \sqrt{\sum_{i,j} |a_{ij}|^2}\)$$。 |
| **谱范数（2-范数）** | $$\(\|A\|_2 = \sigma_{\max}(A)\)$$，最大奇异值。 |
| **1-范数** | $$\(\|A\|_1 = \max_j \sum_i |a_{ij}|\)$$。 |
| **∞-范数** | $$\(\|A\|_\infty = \max_i \sum_j |a_{ij}|\)$$。 |

## 10. 线性代数的经典结果与技巧

| 结果/技巧 | 说明 |
|-----------|------|
| **秩-零化定理** | $$\(\operatorname{rank}A+\operatorname{nullity}A = n\)$$。 |
| **投影与最小化** | 直线、平面等最短距离可通过正交投影求解。 |
| **矩阵不等式** | $$\(\|AB\| \le \|A\|\,\|B\|\)$$（子乘子性质）。 |
| **对称矩阵的正定性** | 若所有特征值 $$\(>0\)$$，则正定；若 $$\(\ge 0\)$$，则半正定。 |
| **范数不等式** | $$\(\|x\|_2 \le \|x\|_1 \le \sqrt{n}\,\|x\|_2\)$$。 |
| **矩阵与线性映射的等价** | 任何线性映射在给定基下可表示为矩阵，矩阵乘法即映射的复合。 |

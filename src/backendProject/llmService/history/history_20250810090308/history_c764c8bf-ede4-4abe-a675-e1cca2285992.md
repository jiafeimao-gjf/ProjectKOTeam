**代码示例 (Python):**

```python
def print_numbers_in_batches(start, end, batch_size):
  """
  分批次打印指定范围内的英文数字。

  Args:
    start: 范围的起始值 (包含)。
    end: 范围的结束值 (包含)。
    batch_size: 每批打印的数字数量。
  """
  for i in range(start, end + 1):
    print(i)
    if (i + 1) % batch_size == 0:
      print()  # 换行，方便阅读

# 设置参数
start_number = 1
end_number = 100
batch_size = 10

# 分批次打印
print_numbers_in_batches(start_number, end_number, batch_size)
```

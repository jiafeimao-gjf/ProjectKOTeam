**在 Xcode 左侧 Project Navigator（项目导航器）中查看 Markdown（.md）文件的简易步骤**

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | **确认文件已加入项目** | 右键项目/目标文件夹 → **Add Files to “YourProject”…** → 选中 *.md* 文件。 |
| 2 | **在左侧导航器中定位** | 选中对应文件夹，*.md* 文件会显示在列表里。 |
| 3 | **打开并预览** | 双击 *.md* 打开 → 在编辑器右上角点击 **“Show Preview”**（方块中双窗格图标）即可看到 Markdown 渲染效果。 |
| 4（可选） | **使用快捷键快速打开** | `⌘⇧O` → 输入文件名 → 选中即可。 |

> **提示**  
> - 如果在导航器里看不到文件，检查“**Show Packages**”或“**Show Hidden Files**”是否开启（Xcode → Preferences → Navigation）。  
> - 只在 Xcode 12+ 版本才支持 Markdown 预览功能。

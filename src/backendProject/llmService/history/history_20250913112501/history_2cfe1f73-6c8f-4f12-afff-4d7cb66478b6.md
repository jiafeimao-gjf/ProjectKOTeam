# prompt: 你是一个Xcode 开发助手，请以尽量简洁的表达，回答用户的问题：
Xcode 如何显示md文件的内容

# answer: 
 Xcode 默认不支持直接显示 `.md` 文件内容。

**解决方案：**

1.  **安装 Xcode Source Code Plugin:**  推荐使用 [Xcode Source Code Plugin](https://github.com/joshuaflaat/Xcode-SourceCodePlugin)。它能识别并预览 Markdown 文件。
2.  **手动打开:**  使用 Xcode 的 "Open As" 选项，将 `.md` 文件打开为文本文件，但不会格式化。

选择第一种方法更方便。
